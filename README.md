## BestTrace路由追踪

```shell
bash <(curl -sL raw.githubusercontent.com/honeok/shell/master/BestTrace.sh)
```

### 参数说明：

脚本支持以下命令行参数来选择不同区域的路由回程：

**默认执行所有区域的路由回程**

- `sz`：深圳三网回程
- `sh`：上海三网回程
- `bj`：北京三网回程
- `cd`：成都三网回程

### 示例用法：

```shell
bash <(curl -sL raw.githubusercontent.com/honeok/shell/master/BestTrace.sh)
```
