## Install

Install dependencies.

```shell
python3 -m pip install --no-cache-dir -r requirements.txt
```

Download Telegram channel config file.

```shell
curl -L https://github.com/yitong2333/proxy-minging/raw/main/config.yaml -o channel.yaml
```

Enjoy!

```shell
python3 main.py
```
