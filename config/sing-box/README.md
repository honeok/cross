<img src="https://avatars.githubusercontent.com/u/83217677" alt="Project S" width="150"/>

This repository uses [233boy's script][2] to install sing-box and then applies a custom configuration.

```shell
bash <(curl -Ls https://fastly.jsdelivr.net/gh/233boy/sing-box@main/install.sh)
```
